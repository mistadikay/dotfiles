### install

Prerequisites:
- Homebrew: http://brew.sh/

```bash
git clone --recursive https://github.com/mistadikay/dotfiles ~/.dotfiles && bash ~/.dotfiles/install.sh
```

### sync

```bash
bash ~/.dotfiles/sync.sh

# or using fish
dsync
```
